# Iconify Starter Registry

This directory contains the definitions for the **Starter Registry**, which ships as a bundled asset with the `iconify_sdk` package.

## Goal
To provide a seamless "out-of-the-box" experience for developers, ensuring the most common icons work instantly without requiring internet access or CLI configuration.

## Registry Contents
- **Material Design Icons (MDI)**: Top 150 icons by usage.
- **Lucide**: Top 100 icons.
- **Tabler Icons**: Top 100 icons.
- **Heroicons**: Complete set (~300 icons).
- **Global Metadata**: Lightweight metadata (name, author, license) for all 200+ Iconify collections.

## Technical Constraints
- **Size Budget**: Under **200KB uncompressed**.
- **Format**: Optimized JSON shards for fast initial parsing.

## Build Process
The data in this directory is generated by the `tools/build_starter_registry.dart` script (to be implemented in Phase 2).
